# Random-Quote-Machine  
***
A javascript App that displays random quotes and its authors dynamically on the UI fetching these quotes from an API server. I used the quote and design api to achieve this. 


## LANGUAGES USED
***

HTML5  
CSS3,
SCSS,
FONT AWESOME,  
JAVASCRIPT/JQUERY,
API 


## CSS FRAMEWORK USED:
